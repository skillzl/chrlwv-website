# chrlwv website

Landing page for chrlwv-tech project written in TypeScript.

## Usage

* Getting Started
```bush
git clone https://github.com/skillzl/chrlwv-website.git
cd chrlwv-website
npm install

```
* Start the project

```bush
npm run dev
```
